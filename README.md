# Ruta 20 — Agencia de Viajes Eje Cafetero

Plataforma web para paquetes turísticos a Armenia, Pereira y Manizales.

## Requisitos
- XAMPP (PHP 8.x + MySQL)
- Navegador moderno

## Instalación
1. Clona el repositorio en `htdocs/ruta20`
2. Copia `.env.example` a `.env` y configura tus credenciales
3. Importa `database/ruta20.sql` en phpMyAdmin
4. Accede a `http://localhost/ruta20`
